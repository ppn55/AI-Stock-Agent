# AI Stock Agent

An AI-powered Taiwan stock analysis assistant.

## Features

- Read holdings from Google Sheets
- Download Taiwan stock prices
- Calculate technical indicators
- Analyze with GPT (Zeabur AI Hub)
- Generate daily HTML reports
- Send email notifications

## Tech Stack

- Python
- Google Sheets
- Zeabur AI Hub
- OpenAI Compatible API
- yfinance
- Pandas
